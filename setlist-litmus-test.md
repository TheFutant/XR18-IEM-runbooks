# Setlist Litmus Test

A shared scoring rubric for evaluating whether a candidate song belongs on the setlist. Score each song 0-2 per category (0 = fail, 1 = partial, 2 = pass). Max score: 14.

**Threshold: 10+ = keeper. 7-9 = maybe / discuss. Under 7 = cut.**

### Scoring guide

- **2** — Clear pass, no hesitation.
- **1** — Partial / conditional (e.g., recognizable to only one generation, groove is good but not great, tempo is close but not quite in range).
- **0** — Clear fail.

### Reminders while scoring

- A song can be *musically great* and still fail this test — that's fine. This is a floor-filler filter, not a taste filter.
- 1-2 songs per set can score in the "maybe" range as deliberate breathers, ballads, or band-favorite wildcards — just don't let them cluster back-to-back.
- If a song only scores well on Recognition or "we love playing it," flag it for discussion rather than auto-including it.
