House Tone — Thick / Warm / Thumping

North star for this project: thick, warm, thumping fingerstyle bass, built from the core path. Loop B textures are parked — available, not featured with this band. The Stax 5-band EQ is owned but off the board for now.
The thick-warm-thump recipe ▶voicing
  • Bass — favor the P (neck) pickup for fundamental thump; dial the J back (it adds growl/bite). Play closer to the neck for a rounder attack.
  • Passive mode is the warm, vintage-thump voice — rounder and fatter. Roll the tone (treble knob) back a touch. Use active only when you need more output/clarity, and keep the treble down.
  • Amp: Brite OFF. Bass slightly up, treble at/below noon, and don't scoop the mids — low-mid body (~200–400 Hz) is what makes it thump in the mix, not just on stage.
  • Compressor ~10:00 for even, punchy thump. Push Gain enough to warm the 12AX7 (Drive still off).
  • SansAmp = warmth engine: Drive 9–10, Blend ~10–11 for harmonic thickness; Bass at/just above noon, Presence low.
  • Harmonic Booster: a touch always-on (Character adds harmonics + mid presence) helps the tone sit thick and consistent — the one case to lean toward always-on over stomp. Your call.
  • Soul Food stays a pick-song tool; a hair fingerstyle adds harmonic warmth if wanted.
Where each trait comes from: thump = pickup choice + low mids + compression. Warmth = passive/tone-down + tube gain + SansAmp. Restraint up top = Brite off + treble down.

Gain Staging — Bass to Room

Mantra: tone with Gain, volume with Master, Drive off. On a combo these were one knob — now they're separate stages.
Active/passive level jump is normal — don't chase it with the amp. If the drop bugs you live, nudge the Yamaha's onboard, not the head.

Pedal Settings

Boss LS-2 — permanent parallel mixer ▶always A+B
Always in A+B Mix — never switch loops. The LS-2 is a fixed parallel mixer: Loop A is a permanent clean anchor summed in at all times, so Loop B can get weird without ever gutting the low end. You change sounds at the individual pedals, not here.
  • Loop A = clean/grit anchor (Soul Food off = clean, on = pick grit). Always present.
  • Loop B = texture, engaged pedal-by-pedal. Its clean fundamental is redundant — let Loop A own the lows. Note: the Muff crossover HPF only rolls off lows when the fuzz is on, so it can't be your always-on control. For a permanent Loop B low roll-off, add a 5-band EQ last in Loop B — cut ~60–80 Hz hard and ~120–150 Hz moderately, mids/highs flat-to-boosted. (Bass Clone X-Over ON also helps hold its own lows.)
  • Balance A/B levels by ear at practice volume — more A for solid, more B for wash.
  • Watch: the MS-60B adds a hair of digital latency when a patch is active. If lows go hollow with a Zoom patch on, that's Loop B vs Loop A phase — set the MS-60B to true bypass when idle and keep Loop B high-passed.

Zoom MS-60B — Loop B, pre-fuzz ▶texture
Current patch idea: Auto Wah → single PEQ → (into chorus/fuzz). PEQ focuses the growl before the Muff — more vocal synth-bass, less fizz.
  • PEQ: Freq 750 Hz (600 if honky, 900 if disappearing), Gain +2–3 dB, medium Q, unity level.
  • If Auto Wah is too quacky, instead cut ~1.2 kHz by −2 dB.
  • Other lanes to try: slow phaser/flanger, synth-bass patch, utility pre-fuzz EQ. Leave normal chorus to the Bass Clone.

Per-Song Modes

Default — fingerstyle

Soul Food OFF · Harmonic Booster OFF · Loop B pedals OFF
Clean supportive bass. LS-2 stays A+B; both loops pass clean; amp + SansAmp are your finished baseline tone.

Pick songs

Soul Food ON · Booster to taste
Warm OD tames pick transient, adds grit and presence for plectrum passages.

Mid-forward / choruses

Harmonic Booster ON
Mid/presence lift (+1–2 dB) so the bass steps forward. Stackable with Soul Food.

Texture / atmosphere (parked this band)

Engage Loop B pedals individually · clean from Loop A always underneath
Octave/chorus/fuzz wash for Cure & Tool moments — on the bench for this project, ready when a song calls for it.
